# Quantum Square
## 游戏说明
- 玩家1：W键向上，S键向下，D键使用道具
- 玩家2：方向键向上向下，M键使用道具
- 空格键暂停游戏，回车键重新开始游戏
## 道具说明
- 加速：使用后可令自己加速
- 飞碟：使用后可令对手减速
- 大魔王：使用后可令对手按键方向反向
- 导弹：在对手后方使用，可使对手被击飞
- 隐身：可使自己隐身，隐身状态下不会被障碍挡住
- 互换：可以与对手交换位置
## inspire
